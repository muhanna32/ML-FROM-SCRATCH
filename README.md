## 📦 What's Inside

This repo is a personal playground for learning Machine Learning by doing. Each notebook takes a single algorithm, breaks it down to its core math, and builds it up from scratch in Python.

🔹 **9 Jupyter Notebooks** — covering supervised, unsupervised, and ensemble methods:
> Linear Regression · Logistic Regression · K-Nearest Neighbors · Decision Tree · Random Forest · Support Vector Machine · Naive Bayes · K-Means Clustering · XGBoost

🔹 **6 Real Datasets** — clean, ready-to-use `.csv` files for training and testing each model.

## 💡 Purpose / Motivation

This repo isn't a library, a framework, or a production tool — it's a **learning journal**.

I started it because I got tired of treating ML algorithms like magic spells. Calling `model.fit()` is easy, but it never answered my real questions: *Why does it work? What's actually happening inside? How does it decide?*

So I decided to open the hood and build each algorithm from scratch — writing the gradient descent loops, splitting the decision nodes, computing the Bayes probabilities — all by hand, one notebook at a time.

The GIF visualizations aren't just for show. Watching a regression line slowly converge, or K-Means clusters settle into place, made these concepts *click* in a way that textbooks never did for me.

If you're on a similar journey — whether you're a student, a self-taught developer, or just someone who's curious about how machines actually "learn" — I hope these notebooks give you that same _aha!_ moment. 🙌
