## рџ’Ў Purpose / Motivation

This repo isn't a library, a framework, or a production tool вЂ” it's a **learning journal**.

I started it because I got tired of treating ML algorithms like magic spells. Calling `model.fit()` is easy, but it never answered my real questions: *Why does it work? What's actually happening inside? How does it decide?*

So I decided to open the hood and build each algorithm from scratch вЂ” writing the gradient descent loops, splitting the decision nodes, computing the Bayes probabilities вЂ” all by hand, one notebook at a time.

The GIF visualizations aren't just for show. Watching a regression line slowly converge, or K-Means clusters settle into place, made these concepts *click* in a way that textbooks never did for me.

If you're on a similar journey вЂ” whether you're a student, a self-taught developer, or just someone who's curious about how machines actually "learn" вЂ” I hope these notebooks give you that same _aha!_ moment. рџ™Њ
